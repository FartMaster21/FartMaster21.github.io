# Project Title
Music You Should Listen To

## Project Description
I made an interface inspired off of music blogs, where I talked about and showed an EP I like.

## Intended User or Player
This interface is for people who are open to all sorts of music. 

## User / Player Goal
The user should be able to see what EP or album is being talked about, as well as the songs in order and my reviews and descriptions of them. Users can also click on links to see
the artist on streaming services. There is also a comment section.

## Inspiration Interfaces
- Lost Songs (Punk-adjacent music blog)
  - Simple layout with design choices that compliment the nature of the blog
- VocaloReview (Vocaloid blog)
   - Not too focused on design, but has detailed reviews on the songs and a review of the album, unlike Lost Songs
- Sophie's Floorboard (Alternative music blog)
   - Alternative music blog that shares many similiarities with the other interfaces mentioned in this list.

## Design Choices
- Layout: Blog style. Vertical, scroll down type situation
- Color: Very heavy on the pinks and reds
- Typography: Trebuchet MS (my goat) too give an early internet/MS paint vibe
- Visual hierarchy: Header on the top with the title. Then EP name, artist name, album cover, and then songs listed below. Then to the right of the album
  cover there's links and a brief description of the synthesizer used on the EP in my project. All the way at the bottom there is a comment section too.
- Feedback states: Buttons for links, and then a comment section where you can type (but not actually post the comment)
- Responsive design: Site doesn't bug out when tab is made smaller and header remains the same size on mobile
